# WokGen Emoji Mode

## What this mode is for

WokGen Emoji generates small-format symbolic assets:

- **Emoji packs** — themed emoji sets (animal, tech, nature, etc.)
- **Reaction sets** — Discord/Slack custom reaction emojis
- **Sticker packs** — expressive character stickers
- **App icon sets** — status icons, notification badges
- **Platform packs** — sized and formatted for Discord, Slack, iOS, Android

## What this mode is NOT for

- Full illustrations → use WokGen Vector
- Brand logos → use WokGen Business
- Game sprites → use WokGen Pixel

## Size guidelines

Emojis are designed for clarity at 16–64px. The engine renders at 256px for
quality, then exports at target sizes. All outputs are PNG with transparent backgrounds.
