# WokGen Vector Mode

## What this mode is for

WokGen Vector generates scalable design assets:

- **SVG icons** — single color, duotone, outline, filled styles
- **Illustration sets** — cohesive character or object illustrations
- **UI component illustrations** — empty states, onboarding, error pages
- **Pattern systems** — repeating geometric or organic patterns
- **Logo marks** — simple scalable symbolic marks

## What this mode is NOT for

- Pixel art or raster game sprites → use WokGen Pixel
- Complex photographic images → use another engine
- React/HTML components → use WokGen UI/UX

## Output formats

SVG (primary), PNG (rasterized at specified size), ZIP (packs)
