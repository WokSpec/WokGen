# WokGen Business Mode

## What this mode is for

WokGen Business generates brand and marketing assets:

- **Logos** — wordmarks, icon marks, combination marks
- **Brand kits** — complete identity system (logo + palette + typography visual)
- **Slide visuals** — pitch deck and presentation images
- **Social banners** — Twitter/X, LinkedIn, Instagram, Facebook sized
- **Web hero images** — full-width landing page hero sections
- **Icons** — professional product and app icons

## What this mode is NOT for

- Pixel art or game sprites → use WokGen Pixel
- SVG icons or scalable vectors → use WokGen Vector  
- React/HTML UI components → use WokGen UI/UX
